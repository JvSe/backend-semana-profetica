DOCS da API

Uma breve documentação sobre a API desenvolvida para a semana profética.





Observações


  • Tudo o que está entre duas chaves, " {{}} ", são variáveis a serem preenchidas.
  • Nas tabelas, tem um campo com "Variáveis". Esse nome não pode mudar, se estiver diferente desse nome, dará erro no momento de inserir ou buscar alguma informação no banco de dados.



    • Tabela de Rotas da API
    URLs Função da URL Status
    https://backend-semana-profetica.herokuapp.com/user/login Rota para realizar o login
    https://backend-semana-profetica.herokuapp.com/user/cadastro Rota para criar um cadastro de usuário
    https://backend-semana-profetica.herokuapp.com/decretos Rota para buscar uma lista com todos os decretos

    Informações das Rotas

    Aqui tem as informações necessárias para poder utilizar das rotas corretamente.






    Rota para realizar o login no aplicativo




    Esta rota é responsável por realizar o login do usuário no aplicativo. Para sua utilização sem erros, é necessário que o "email" e a "senha" do usuário sejam passados pela tag "body" da aplicação e a requisição seja através do método /GET.

    As informações para cadastro podem ser conferidas na tabela abaixo:



    Tabela de variáveis para realizar o login
    Variável Descrição Tipo do Atributo
    email Variável que recebe o email do usuário String
    senha Variável que recebe a senha do usuário String



    Rota para realizar o cadastro do usuário no aplicativo




    Esta rota é responsável por realizar o cadastro do usuário no aplicativo. Para sua utilização sem erros, é necessário que as informações do usuário sejam passados pela tag "body" da aplicação e a requisição seja através do método /POST.

    As informações para cadastro podem ser conferidas na tabela abaixo:



    Tabela de variáveis para realizar o cadastro de usuário
    Variável Descrição Tipo do Atributo
    nome Variável que recebe o nome da pessoa cadastrada String
    email Variável que recebe o email da pessoa cadastrada String
    senha Variável que recebe senha de login da pessoa cadastrada String
    endereco Variável que recebe o endereco da pessoa cadastrada String
    dataAniversario Variável que recebe a data de aniversário da pessoa cadastrada String
    telefone Variável que recebe o telefone da pessoa cadastrada String
    imgPerfil ESSE AINDA NAO ESTA IMPLEMENTADO



    Rota para buscar todos os decretos




    Esta rota é responsável por realizar a busca de todos os decretos no BD. Para sua utilização sem erros é necessário que a requisição seja através do método /GET.




    Rota para realizar o cadastro de um decreto no aplicativo




    Esta rota é responsável por realizar o cadastro de um decreto no aplicativo. Para sua utilização sem erros, é necessário que as informações do decreto sejam passados pela tag "body" da aplicação e a requisição seja através do método /POST.

    As informações para cadastro podem ser conferidas na tabela abaixo:



    Tabela de variáveis para realizar o cadastro de um decreto
    Variável Descrição Tipo do Atributo
    titulo Variável que recebe o titulo do decreto String
    meusPedidos Variável que recebe a lista de pedidos do decreto Array
    pedidosPorAlguem Variável que recebe os pedidos do decreto direcionados para outra pessoa. Essa variável recebe um objeto com: nomePessoa, pedidos. Object
    user Variável que recebe o id do usuário String


    Rota para buscar todos as informações da tela HOME




    Esta rota é responsável por realizar a busca de todas as informações da tela HOME no BD. Para sua utilização sem erros é necessário que a requisição seja através do método /GET.




    Rota para realizar o cadastro de uma informação para a tela HOME no aplicativo




    Esta rota é responsável por realizar o cadastro de uma informação para tela HOME no aplicativo. Para sua utilização sem erros, é necessário que as informações sejam passados pela tag "body" da aplicação e a requisição seja através do método /POST.

    As informações para cadastro podem ser conferidas na tabela abaixo:



    Tabela de variáveis para realizar o cadastro de um decreto
    Variável Descrição Tipo do Atributo
    titulo Variável que recebe o titulo do decreto String
    descricao Variável que recebe a descricao da informação String
    postagem Variável que recebe as informações da postagem String
    imgPostagem ESSE AINDA NAO ESTA IMPLEMENTADO String
    autor Variável que recebe o nome do autor da publicação String
    imgAutor ESSE AINDA NAO ESTA IMPLEMENTADO String
